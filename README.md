# PTS-Oman

Basic Node.js + Express API structure.

## Folder structure

- `src/app.js` - main app setup
- `src/server.js` - server start file
- `src/routes/api.routes.js` - API route definitions
- `src/controllers/api.controller.js` - API request handling
- `src/services/api.service.js` - service/business logic
- `src/middlewares/error.middleware.js` - centralized error handler
- `src/config/config.js` - environment config

## Run locally

1. Install dependencies:
   - `npm install`
2. Start server:
   - `npm start`

Server runs on `http://localhost:3000` by default.
